# QA-Manual-Testing-Ecommerce
This project demonstrates manual testing of an E-Commerce web application, focusing on core functionalities such as user authentication, product browsing, cart management, and checkout process.  The objective was to validate functional requirements, identify defects, and ensure a smooth user experience.
